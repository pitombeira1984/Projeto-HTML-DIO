Introdução ao HTML

(Voltar)

HTML (HyperText Markup Language) é a linguagem de marcação padrão para criar páginas web.

Ele estrutura o conteúdo e define como os elementos devem ser exibidos no navegador.

Formulário em HTML

(Voltar)
Um formulário HTML é uma seção de uma página web que permite aos usuários inserir e enviar dados para um servidor web.

Os formulários são usados para coletar informações dos usuários, como nome, email, senha, etc.

Os elementos de formulário incluem campos de texto, botões, caixas de seleção, botões de opção, listas suspensas e muito mais.

Os formulários são essenciais para interações dinâmicas em sites, como cadastro, login, comentários e pesquisas.

Para criar um formulário, usamos a tag <form> e dentro dela, adicionamos os elementos de entrada necessários.

Formatando e Estruturando HTML

(Voltar)

HTML permite formatar e estruturar o conteúdo de uma página web usando tags.

As tags são usadas para definir elementos como títulos, parágrafos, listas, links, imagens e muito mais.

As tags HTML são compostas por uma abertura e um fechamento, como <tag> e </tag>.

Algumas tags são auto-fechadas, como <br> e <img>, que não requerem uma tag de fechamento.

As tags HTML podem ter atributos que fornecem informações adicionais sobre o elemento, como id, class, style, etc.

A tag <div> em HTML é usada para definir uma divisão ou seção em um documento HTML.

Utilize tags como <strong>, <em>, <h1> a <h6> para formatar e estruturar seu conteúdo.

Imagens e Mídia c/ HTML

(Voltar)

Para colocar uma única imagem em uma página da web, usamos o elemento .
Isso é um elemento vazio (quer dizer que não possui conteúdo de texto ou tag de fechamento) que requer no mínimo um atributo para ser útil — src (às vezes pronunciado como seu título completo, source).

O atributo src especifica o caminho para a imagem que queremos exibir.
O atributo alt fornece uma descrição alternativa da imagem, que é importante para acessibilidade e SEO.

Para adicionar vídeos, usamos o elemento <video> com o atributo controls para permitir que os usuários reproduzam, pausem e ajustem o volume do vídeo.

O elemento <video> pode conter múltiplas fontes de vídeo usando a tag <source>, permitindo que o navegador escolha o formato mais adequado.

Tabelas c/ HTML

(Voltar)

As tabelas HTML são usadas para mostrar dados tabulares com linhas e colunas.
A decisão de usar um <table> precisa ser baseada no conteúdo que você está apresentando e nas necessidades dos usuários em relação a esse conteúdo.

Uma tabela HTML é composta por várias tags, incluindo <table>, <tr> (linha), <th> (cabeçalho) e <td> (célula de dados).

As tabelas podem ser usadas para exibir informações como listas de produtos, resultados de pesquisas, dados financeiros e muito mais.

As tabelas podem ser estilizadas usando CSS para melhorar a aparência e a legibilidade.

HTML Semântico

(Voltar)

HTML semântico é uma abordagem que utiliza tags HTML para descrever o significado do conteúdo, melhorando a acessibilidade e a SEO.

Tags semânticas incluem <header>, <nav>, <main>, <article>, <section>, <aside>, <footer>, entre outras.

O uso de HTML semântico ajuda os mecanismos de busca a entenderem melhor o conteúdo da página, melhorando a indexação e a visibilidade nos resultados de pesquisa.

Além disso, melhora a acessibilidade para usuários com deficiências, pois leitores de tela podem interpretar melhor o conteúdo semântico.
